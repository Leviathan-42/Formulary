Point-Slope Form

What is Point-Slope Form?

Point-slope form is one of the three main forms for writing linear equations. It's especially useful when you know a point on the line and the slope of the line.

$$y - y_1 = m(x - x_1)$$

Where:

  • \((x_1, y_1)\) is any point on the line
  • \(m\) is the slope of the line
  • \((x, y)\) represents any other point on the line

When to Use Point-Slope Form

Use point-slope form when you're given:

  • One point on the line AND the slope
  • Two points on the line (calculate slope first, then use one point)
  • A graph where you can identify a point and count the slope

Understanding the Slope

The slope \(m\) represents the rate of change (steepness) of the line:

$$m = \frac{\text{rise}}{\text{run}} = \frac{y_2 - y_1}{x_2 - x_1}$$

  • Positive slope: Line rises from left to right (uphill)
  • Negative slope: Line falls from left to right (downhill)
  • Zero slope: Horizontal line (\(y = c\))
  • Undefined slope: Vertical line (\(x = c\))

Step-by-Step: Writing Equations

Example 1: Write the equation with slope \(m = 3\) through point \((2, 5)\)

$$\text{Step 1: Identify values} \quad m = 3, \; x_1 = 2, \; y_1 = 5$$ $$\text{Step 2: Substitute into formula} \quad y - 5 = 3(x - 2)$$ $$\text{Step 3: (Optional) Convert to slope-intercept}$$ $$y - 5 = 3x - 6$$ $$y = 3x - 1$$

Example 2: Find equation through points \((1, 4)\) and \((3, 10)\)

$$\text{Step 1: Find slope} \quad m = \frac{10 - 4}{3 - 1} = \frac{6}{2} = 3$$ $$\text{Step 2: Use either point (we'll use } (1, 4))$$ $$y - 4 = 3(x - 1)$$ $$\text{Or using } (3, 10): \quad y - 10 = 3(x - 3)$$

Note: Both equations represent the same line!

Converting Between Forms

Point-Slope to Slope-Intercept: \(y = mx + b\)

  • Distribute the slope on the right side
  • Add \(y_1\) to both sides to isolate \(y\)

$$y - 3 = 2(x - 4)$$ $$y - 3 = 2x - 8$$ $$y = 2x - 5$$

Common Mistakes to Avoid

  • ‚ùå Forgetting the negative sign: It's \(y - y_1\), not \(y + y_1\)
  • ‚ùå Mixing up \(x\) and \(y\) coordinates when substituting
  • ‚ùå Incorrectly calculating slope (remember: \(\frac{\Delta y}{\Delta x}\))
  • ‚ùå Distributing incorrectly: \(3(x - 2) = 3x - 6\), not \(3x - 2\)

Practice Problems

Try these on your own:

  1. Write equation with slope \(-2\) through \((3, 7)\)
  2. Find equation through \((-1, 5)\) and \((2, -4)\)
  3. Convert \(y - 6 = -\frac{1}{2}(x + 4)\) to slope-intercept form

Standard Form

What is Standard Form?

Standard form is another way to write linear equations. It's especially useful for quickly finding x and y intercepts and working with systems of equations.

$$Ax + By = C$$

Where:

  • \(A\), \(B\), and \(C\) are integers (no fractions or decimals)
  • \(A\) should be positive (if negative, multiply entire equation by -1)
  • \(A\) and \(B\) cannot both be zero
  • Greatest Common Factor (GCF) of \(A\), \(B\), and \(C\) should be 1

Why Use Standard Form?

  • Finding intercepts: Easy to find where line crosses axes
  • Systems of equations: Simplifies elimination method
  • Integer coefficients: Easier to work with whole numbers
  • Real-world problems: Some applications naturally fit this form

Finding Intercepts from Standard Form

Given equation: \(3x + 4y = 12\)

X-intercept (where line crosses x-axis, so \(y = 0\)):

$$3x + 4(0) = 12$$ $$3x = 12$$ $$x = 4$$ $$\text{X-intercept: } (4, 0)$$

Y-intercept (where line crosses y-axis, so \(x = 0\)):

$$3(0) + 4y = 12$$ $$4y = 12$$ $$y = 3$$ $$\text{Y-intercept: } (0, 3)$$

Converting TO Standard Form

From Slope-Intercept Form: \(y = 2x + 3\)

$$y = 2x + 3$$ $$-2x + y = 3 \quad \text{(move x term to left)}$$ $$2x - y = -3 \quad \text{(multiply by -1 to make A positive)}$$

From Point-Slope Form: \(y - 4 = 3(x - 2)\)

$$y - 4 = 3(x - 2)$$ $$y - 4 = 3x - 6 \quad \text{(distribute)}$$ $$y = 3x - 2 \quad \text{(simplify)}$$ $$-3x + y = -2$$ $$3x - y = 2 \quad \text{(standard form)}$$

With Fractions: \(y = \frac{2}{3}x + 4\)

$$y = \frac{2}{3}x + 4$$ $$3y = 2x + 12 \quad \text{(multiply everything by 3)}$$ $$-2x + 3y = 12$$ $$2x - 3y = -12 \quad \text{(standard form)}$$

Converting FROM Standard Form

To Slope-Intercept Form: Solve for \(y\)

$$4x + 2y = 8$$ $$2y = -4x + 8 \quad \text{(isolate y term)}$$ $$y = -2x + 4 \quad \text{(divide by 2)}$$

Graphing from Standard Form

Method 1: Using Intercepts

  1. Find x-intercept (set \(y = 0\))
  2. Find y-intercept (set \(x = 0\))
  3. Plot both points and draw line through them

Method 2: Convert to Slope-Intercept

  1. Solve for \(y\) to get \(y = mx + b\) form
  2. Plot y-intercept \(b\)
  3. Use slope \(m\) to find other points

Common Mistakes

  • ‚ùå Leaving fractions: \(\frac{1}{2}x + y = 3\) ‚Üí Should multiply by 2: \(x + 2y = 6\)
  • ‚ùå Negative \(A\): \(-2x + 3y = 5\) ‚Üí Should be \(2x - 3y = -5\)
  • ‚ùå Not simplifying GCF: \(4x + 6y = 10\) ‚Üí Should be \(2x + 3y = 5\)
  • ‚ùå Sign errors when rearranging terms

Special Cases

Horizontal Lines: \(y = 5\) becomes \(0x + 1y = 5\) or simply \(y = 5\)

Vertical Lines: \(x = 3\) becomes \(1x + 0y = 3\) or simply \(x = 3\)

Practice Problems

  1. Convert \(y = -\frac{3}{4}x + 5\) to standard form
  2. Find the x and y intercepts of \(5x - 2y = 10\)
  3. Convert \(2x + 6y = 18\) to simplest standard form
  4. Convert \(3x - 4y = 12\) to slope-intercept form

Parallel & Perpendicular Lines

Understanding Parallel Lines

Parallel lines are lines in the same plane that never intersect, no matter how far they extend. They maintain the same distance apart at all points.

Key Rule: Parallel lines have identical slopes

$$\text{If } L_1: y = m_1x + b_1 \text{ and } L_2: y = m_2x + b_2$$ $$\text{Then } L_1 \parallel L_2 \text{ if and only if } m_1 = m_2 \text{ and } b_1 \neq b_2$$

Example 1: Verify if these lines are parallel

  • Line 1: \(y = 3x + 7\) ‚Üí slope = 3
  • Line 2: \(y = 3x - 2\) ‚Üí slope = 3
  • ‚úÖ Same slope, different y-intercepts ‚Üí Parallel!

Example 2: Write equation parallel to \(y = -2x + 5\) through point \((3, 4)\)

$$\text{Step 1: Identify slope} \quad m = -2$$ $$\text{Step 2: Use point-slope form with the point } (3, 4)$$ $$y - 4 = -2(x - 3)$$ $$y - 4 = -2x + 6$$ $$y = -2x + 10$$

Understanding Perpendicular Lines

Perpendicular lines intersect at a 90¬∞ angle (right angle). Think of the corner of a square or the intersection of a cross.

Key Rule: Perpendicular slopes are negative reciprocals

$$L_1 \perp L_2 \text{ if } m_1 \cdot m_2 = -1$$ $$\text{or equivalently } m_2 = -\frac{1}{m_1}$$

Finding Negative Reciprocals

To find the negative reciprocal of a slope:

  1. Flip the fraction (reciprocal)
  2. Change the sign (negative)
Original Slope Perpendicular Slope
\(m = 2\) or \(\frac{2}{1}\) \(m_\perp = -\frac{1}{2}\)
\(m = -3\) or \(-\frac{3}{1}\) \(m_\perp = \frac{1}{3}\)
\(m = \frac{3}{4}\) \(m_\perp = -\frac{4}{3}\)
\(m = -\frac{5}{2}\) \(m_\perp = \frac{2}{5}\)

Example 3: Verify if lines are perpendicular

  • Line 1: \(y = 4x + 1\) ‚Üí \(m_1 = 4\)
  • Line 2: \(y = -\frac{1}{4}x + 7\) ‚Üí \(m_2 = -\frac{1}{4}\)
  • Check: \(4 \times (-\frac{1}{4}) = -1\) ‚úÖ Perpendicular!

Example 4: Write equation perpendicular to \(y = \frac{2}{3}x - 4\) through \((-6, 5)\)

$$\text{Step 1: Find perpendicular slope} \quad m_\perp = -\frac{3}{2}$$ $$\text{Step 2: Use point-slope form}$$ $$y - 5 = -\frac{3}{2}(x - (-6))$$ $$y - 5 = -\frac{3}{2}(x + 6)$$ $$y - 5 = -\frac{3}{2}x - 9$$ $$y = -\frac{3}{2}x - 4$$

Special Cases

Horizontal and Vertical Lines:

  • Horizontal line: \(y = c\) has slope \(m = 0\)
  • Vertical line: \(x = c\) has undefined slope
  • Horizontal and vertical lines are always perpendicular to each other

Quick Reference Table

Relationship Slope Condition Example
Parallel \(m_1 = m_2\) \(y = 5x + 2\) and \(y = 5x - 7\)
Perpendicular \(m_1 \cdot m_2 = -1\) \(y = 5x + 2\) and \(y = -\frac{1}{5}x + 3\)
Neither Other slopes \(y = 2x + 1\) and \(y = 3x - 4\)

Common Mistakes

  • ‚ùå Confusing parallel and perpendicular slope rules
  • ‚ùå Forgetting to flip AND change sign for perpendicular slopes
  • ‚ùå Thinking parallel lines must have same y-intercept (they must be different!)
  • ‚ùå Multiplying slopes instead of checking if product = -1

  2. Write the equation parallel to \(2x + 3y = 12\) through point \((0, -4)\)
  3. Write the equation perpendicular to \(y = 4x - 1\) through point \((8, 2)\)
  4. Find the perpendicular slope to \(m = -\frac{7}{3}\)

Linear Inequalities

What are Linear Inequalities?

A linear inequality is like a linear equation, but instead of an equals sign, it uses an inequality symbol. Instead of one specific answer, the solution is a range of values.

Inequality Symbols

Symbol Meaning Example In Words
\(<\) Less than \(x < 5\) x is less than 5
\(>\) Greater than \(x > 5\) x is greater than 5
\(\leq\) Less than or equal to \(x \leq 5\) x is at most 5
\(\geq\) Greater than or equal to \(x \geq 5\) x is at least 5

Solving One-Variable Inequalities

Solve inequalities the same way as equations, with ONE CRITICAL RULE:

‚ö†Ô∏è When multiplying or dividing by a NEGATIVE number, FLIP the inequality sign!

Example 1: Solve \(3x - 2 \geq 7\) (no negatives)

$$3x - 2 \geq 7$$ $$3x \geq 9 \quad \text{(add 2 to both sides)}$$ $$x \geq 3 \quad \text{(divide by 3, sign stays same)}$$

Solution: All numbers greater than or equal to 3

Example 2: Solve \(-2x + 5 > 11\) (dividing by negative!)

$$-2x + 5 > 11$$ $$-2x > 6 \quad \text{(subtract 5)}$$ $$x < -3 \quad \text{(divide by -2, FLIP the sign!)}$$

Solution: All numbers less than -3

Example 3: Solve \(\frac{x}{4} - 3 \leq 2\)

$$\frac{x}{4} - 3 \leq 2$$ $$\frac{x}{4} \leq 5 \quad \text{(add 3)}$$ $$x \leq 20 \quad \text{(multiply by 4)}$$

Graphing Two-Variable Inequalities

For inequalities like \(y < 2x + 1\) or \(3x + 4y \geq 12\):

Step 1: Graph the Boundary Line

  • Use dashed line for \(<\) or \(>\) (not included)
  • Use solid line for \(\leq\) or \(\geq\) (included)
  • Convert to \(y = mx + b\) or use intercepts

Step 2: Choose a Test Point

  • Usually test \((0, 0)\) if it's not on the line
  • Substitute into original inequality
  • If TRUE ‚Üí shade the side with the test point
  • If FALSE ‚Üí shade the opposite side

Example 4: Graph \(y > 2x - 3\)

$$\text{Step 1: Graph } y = 2x - 3 \text{ with a DASHED line}$$ $$\text{Step 2: Test } (0, 0): \quad 0 > 2(0) - 3 \rightarrow 0 > -3 \; \checkmark \text{ TRUE}$$ $$\text{Step 3: Shade the region containing } (0, 0)$$

The solution is all points above the dashed line.

Example 5: Graph \(3x + 2y \leq 6\)

$$\text{Step 1: Find intercepts or solve for y}$$ $$2y \leq -3x + 6$$ $$y \leq -\frac{3}{2}x + 3$$ $$\text{Step 2: Graph with SOLID line (} \leq \text{)}$$ $$\text{Step 3: Test } (0, 0): \quad 3(0) + 2(0) \leq 6 \rightarrow 0 \leq 6 \; \checkmark$$ $$\text{Step 4: Shade region with } (0, 0)$$

Understanding the Shading

  • \(y >\) or \(y \geq\): Shade above the line
  • \(y <\) or \(y \leq\): Shade below the line
  • Dashed line: Points on the line are NOT solutions
  • Solid line: Points on the line ARE solutions

Compound Inequalities

Combining two inequalities:

"And" Compound: \(x > 2\) AND \(x < 7\) written as \(2 < x < 7\)

  • Solution must satisfy BOTH conditions
  • Solution: \(x\) between 2 and 7

"Or" Compound: \(x < -1\) OR \(x > 5\)

  • Solution satisfies AT LEAST ONE condition
  • Solution: \(x\) less than -1 OR greater than 5

Real-World Applications

Example 6: You have $50 to spend on lunch. Sandwiches cost $8 and drinks cost $3. Write an inequality for possible combinations.

$$\text{Let } s = \text{sandwiches}, \; d = \text{drinks}$$ $$8s + 3d \leq 50$$

Graph this to find all possible combinations you can buy!

Common Mistakes

  • ‚ùå Most Common: Forgetting to flip sign when dividing/multiplying by negative
  • ‚ùå Using solid line for \(<\) or \(>\) (should be dashed)
  • ‚ùå Shading the wrong side of the line
  • ‚ùå Not testing a point to verify which side to shade
  • ‚ùå Treating inequality like an equation with only one solution

Quick Reference: Flipping the Sign

You MUST flip the inequality when:
‚úì Multiplying both sides by a negative number
‚úì Dividing both sides by a negative number

You DON'T flip when:
‚úó Adding or subtracting anything
‚úó Multiplying or dividing by a POSITIVE number
